## Brand Guidelines

### Primary Colors
| Token | OKLCH | Hex | Usage |
|-------|-------|-----|-------|
| Primary | `oklch(0.48 0.22 15)` | #B83232 | WCAG AA text |
| Primary Vibrant | `oklch(0.68 0.22 15)` | #FF6B6B | Decorative |
| Accent Purple | `oklch(0.65 0.18 290)` | #9B59B6 | Referral/Special |

### Typography
- **UI Text:** Outfit (Google Fonts)
- **Display:** Playfair Display
- **Monospace:** JetBrains Mono

### Key Metrics
- Border Radius: `20px` standard
- Touch Target: `44px` minimum
- Color Contrast: `5.1:1` (WCAG AA)

### Nano Banana MCP Server
```bash

# Tools: generate_image, edit_image, continue_editing
# API Key: GEMINI_API_KEY env variable
```

### Design Generator Skill
```bash

# Types: screen, component, icon, illustration, mockup
```
